# Todo App

これは、React・Laravel・LiveWire・Tailwind CSS を使用して作成されたシンプルなToDoアプリケーションです。

---

### 必要条件
- Node.js (推奨: v18以上)
- npm または yarn
- PHP (推奨: v8以上)
- MySQL または SQLite



---

## 🛠️ 使用技術
- **フロントエンド:** React, React Router, React Query, Tailwind CSS, Material UI
- **バックエンド:** Laravel 10, LiveWire
- **データベース:** MySQL
- **その他:** Axios, Laravel Mix, Webpack

---
